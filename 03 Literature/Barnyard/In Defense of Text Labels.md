# In Defense of Text Labels

Interfaces that only have icons are worse than those that will use text labels, since icons don't translate meaning as clearly as text. Using text removes ambiguity of purpose and makes learning new interfaces simpler.

> [!quote]
> The next time you’re tempted to create yet another icon, or to remove text labels, remember: the most elegant solution isn’t always the one that _looks_ simple — it’s the one that makes communication and understanding _feel_ simple.
